Des idées de supports de peinture originaux

Voici une petite sélection de supports originaux autre que des châssis entoilés.

Les supports

Plastique

Le plastique est une matière lisse est peu poreuse il est donc préférable avant de peindre d'utiliser une sous couche de peinture via une bombe de peinture pour aider les prochaines couches à tenir sur le support.

La peinture conseillée pour le plastique est : la peinture acrylique

L'ardoise

L'ardoise est une roche avec un grain très fin sur laquel il est très agréable de peindre.

Les peintures conseillées pour l'ardoise sont : la peinture acrylique ou la peinture à l'huile

Le papier buvard

Le papier buvard est un papier plus poreux qui permet d'absorber plus facilement les liquides. Le papier buvard est très utilisé lors de l'utilisation de l'aquarelle.

La peintures conseillée pour le papier est : la peinture aquarelle

Le bois

Le bois est une matière très facile à utilisée, elle offre la possibilité de peindre avec presque toutes les peintures.

Les peintures conseillées pour le bois sont : la peinture aquarelle, la peinture acrylique, la peinture à l'eau(gouache), peinture à l'huile etc...
